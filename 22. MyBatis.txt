1. MyBatis 개요
    1) 자바 코드 안의 SQL 구문을 별도의 xml로 분리한다.
    2) 자바 코드로부터 SQL문을 분리해서 관리
    3) 매개변수 설정과 쿼리 결과물을 읽어오는 코드를 제거
    4) 작성할 코드가 줄어서 생선상 향상, 유지보수 관리

    5) SQL 별도의 파일로 분리해서 관리 
        - 객체-SQL 사이의 파라미터 Mapping 작업을 자동으로 해준다.
    6) JPA(Java Persistence Api)나 Hibernate 등 DB 프로그래밍과
       병행된 개발 기술이다.
    7) 퍼시턴스 프레임워크
        - JDBC의 모든 기능을 MyBatis가 대부분 제공한다.
        - 복잡한 JDBC 코드를 걷어내며 슬림한 소스코드 유지가 가능하다.
        - SQL에 변경이 있을 때마다 자바 코드를 수정하거나 컴파일 하지 않아도 된다.
